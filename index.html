O que é Acessibilidade

A Acessibilidade digital permite que todas as pessoas possam utilizar sites, aplicativos e sistemas com mais facilidade

Por que é importante?

Pessoas com baixa visão podem precisar de letras maiores, bom contraste e conteúdos bem organizados para navegar melhor.

Dicas de acessibilidade

  • Utilizar letras legíveis.
  • Usar cores com bom contraste.
  • Organizar o conteúdo em títulos.
  • Evitar textos muito pequenos.

Projeto desenvolvido com HTML, CSS e JavaScript
